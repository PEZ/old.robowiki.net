How does it move?

How does it fire?

Has a variable length symbolic pattern matcher (100.04% PMC at the last check). Maximum search length 128 scans, 5000 scan history.

How does it dodge bullets?

Although teh firing of a bullet is recorded it is not directly dodged (i don't think...)

What does it save between matches?

The scan history, in a very similar way to how Pez did in LeachPMC (i had problems doing it any other way).
