How does it move?

WaveSurfing with DynamicClustering. I added some pretty interesting features to it such as multiple weighting systems and rolling averages. Besides the danger calculation, it is identical to WaveSerpent's movement.

How does it fire?

The same as WaveSerpent: a dynamic clustering gun with scans segmented on velocity and acceleration.

How does it dodge bullets?

WaveSurfing.

How does the melee strategy differ from one-on-one strategy?

There's currently no melee strategy.

What does it save between rounds and matches?

It saves gun and surfing data between rounds, nothing between matches.

Where did you get the name?

The Hydra is an almost unkillable beast from Greek mythology.
