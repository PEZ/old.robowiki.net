Description
SledgeHammer is rambot of codesize "nano". It uses advanced targeting and movement, but stays still a nano.

One-on-one only

http://ever-rage.narod.ru/robowiki/demetrix.nano.SledgeHammer_0.22.jar

Targeting
It uses simple Linear Targeting (LT), which packed into "nano".

Movement
Ram, ram and ram again... But also bot has a little trick - it cuts the way to the enemy.
Maybe idea is not so new, but I named this as "Cut the way" technique.
