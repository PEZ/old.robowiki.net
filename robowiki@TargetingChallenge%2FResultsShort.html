TC Results Short (100 rounds) version

Searhing the bug in the targeting of GloomyDark 0.5* doesn't progress any... To test things I run a shorter version of the TargetingChallenge. Since the RoboRumble@Home favours fast learning guns I thought this might be interesting for you others too. Therefore this page. Apart from the shorter battles - same rules as in the extended version. I am experimenting some with what to save and am currently trying to do what Paul Evans called a CribSheet. Thus I run the "Short TC" for two seasons and should, if things work, have better results in season 2.

Please feel invited to join this challenge. To help you get the results up on this page quicker I have made a "Targeting Challenge Calculator" for us: /Calculator

-- PEZ

Nice test, only takes quite a while to run... And two conclusions:

  • the PM-gun is not too bad: it's Fenrir's movement that places it halfway the rankings.
  • saving data does not really help a lot (even seems to degrade the performance of the gun in some cases). Now running 500 battles to check if the differences between seasons 1 and 2 are real or due to 'noise' --Loki
