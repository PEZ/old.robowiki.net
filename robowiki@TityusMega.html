How does it move?

It's a random oscillation movement. Derived from Tityus orbit movement.

How does it fire?

A simple, segmented GuessFactorTargeting gun.

How does it dodge bullets?

It just tries to move so unpredictable as it can.

How does the melee strategy differ from one-on-one strategy?

This is a 1v1 bot for now.

How does it select a target to attack/avoid in melee?

This is a 1v1 bot for now.

What does it save between rounds and matches?

It's guess factors are saved between rounds.

Where did you get the name?

This bot is the result of me not fitting some of my ideas into the MiniBot Tityus.

Not yet.
