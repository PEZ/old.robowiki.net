DuelistNanoMelee performs well vs. microbots and minibots, and wins among the top nanobots when it faces them, but it doesn't do very well in battles with nanobots that weren't designed for melee. The source code for DuelistNanoMelee is available. I ask that you acknowledge any code or ideas used, and make a robot that is uniquely yours - not just a slightly modified copy of mine.

--David Alves
