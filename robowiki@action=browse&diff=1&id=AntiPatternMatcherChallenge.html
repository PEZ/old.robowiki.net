Anti Pattern Matcher Challenge



A part of the MovementChallenge. This is the setup:
* PatternMatcherBot (currently a stationary Ender, available at http://www.robocoderepository.com/BotDetail.jsp?id=1568)
* The Challenger (your bot)
* Arena size 800 x 600
* 500 rounds
* Everything else set to Robocode defaults
Additionally:
# Challenger is not allowed to fire any bullets at all
# The ratio "Challenger score / PatternMatcherBot score" * 100 is the Challengers /APMCIndex

A part of the MovementChallenge. This is the setup:
