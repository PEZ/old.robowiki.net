Directions

Here's how you run a MeleeSurfingChallenge:

  • Get the reference bots here: https://davidalves.net/robocode/wiki.challenge.MeleeSurfingChallengeTeams_1.0.jar
  • Team A fires with HeadOnTargeting, Team B fires with LinearTargeting. Source code for both bots is included.
  • Run 1000 rounds against each of them
  • Arena size 1000 x 1000
  • Everything else set to Robocode defaults
  • You are not allowed to fire any bullets

Your bot's score is average amount of life you had at the end of each round. 100 means that the reference teams never hit you. You calculate your score like this: 

 100 - (reference team's bullet damage score / 1000)
