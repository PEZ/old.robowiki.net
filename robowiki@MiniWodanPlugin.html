A FloodGrapher Plugin for the bvh.mini.Wodan file-format.

You can download it at https://www.robocoderepository.com/BotDetail.jsp?id=2076

Installation:

  • this plugin supports the file-format used by bvh.mini.Wodan v0.256 (bvh.mini.Wodan can be obtained at https://robocoderepository.com/BotDetail.jsp?id=2064)
  • unzip the contents of this file in your ..\robocode\robots-directory.
  • Add the following line to the file ..\robocode\robots\kawigi\resoruce\floodgrapher.ini
fileformat=bvh.plugin.miniWodanFormat?

or use FloodGrapher's plugin manager: open up FloodGrapher, go to the "Plugins" tab, hit the add button under the file format list, and type in "bvh.plugin.miniWodanFormat?" in the box.

see also: https://robowiki.net/perl/robowiki?FloodGrapher
