What's special about it?

How competitive is it?

It is a fairly good HaikuBot, and would probably make an effective FemtoBot as well.

How does it move?

It moves clockwise around the battlefield, staying near the walls. It turns when it nears a corner.

How does it fire?

It spins its gun around by PI/16 radians per turn, which allows it to cover the whole battlefield quickly while still retaining some accuracy.

How does the melee strategy differ from one-on-one strategy?

No difference.

What does it save between rounds and matches?

Nothing.

Where did you get the name?

It is a wall-smoothing haiku bot. Hence, it is SmoothHaiku.

Can I use your code?

Yep.
