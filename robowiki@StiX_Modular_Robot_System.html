StiX Modular Robot Architecture

Robo Home | Changes | Preferences | AllPages
StiX Modular Robot Architecture is a modular robot template system, in which you have one main robot class, which then calls upon 3 other classes to handle the navigation, weapons, and sensors.

I'll upload the StiX system soon, so you can all check it out and see if you like it.. ShizzleStiX is based on Stix 0.3, so if you go find it on robocode repository you can get the source from the JAR.
