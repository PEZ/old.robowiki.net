Features:

  • 1v1 Specialist
  • Coded with ModeSelection as the main structure
  • Multiple targeting modes
  • Multiple move modes
  • Multiple radar modes
  • Saves data on other bots to improve performance
  • Uses VirtualBullets to select best firing mode (Thanks to P.E and R.H for the ideas)
