installation:

you have to download Java3D OpenGL/Direct3D?.

1. install Robocode

2. install RobocodeGLV014

3. copy all files inside robocode3D.zip into the robocode folder (like RobocodeGLV014)

now you should run TerrainConfig?.bat and set max hill to 0 (JSlider) and don't forget to save. for tests without Robocode run Metavetsum.bat if it works try Robocode3D.bat if you start a new Battle then pause it, and wait until all robots are visible

http:/robocode/uploads/bienator/robocode3D.zip

I have tryed it 2 times, I think it should work if you follow the order. -- Bienator
