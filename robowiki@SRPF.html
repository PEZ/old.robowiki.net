The StatistRobot Plugin for FloodGrapher.

You can download it here: http://www.robocoderepository.com/BotDetail.jsp?id=2072

After add this Plugin you can see the graph of data gathered by StatistRobot.

Install

1.This Plugin need StatistRobot's Support,So you must download StatistRobot first,it's here:http://robocoderepository.com/BotDetail.jsp?id=1494

2.When download it,exact the two files to same dictionary,commend "your robocode dir\robots\".

3.Add this word "fileformat=org.robochina.plugin.StatistRobotFormat" into your floodgrapher.ini.

It also works to configure it in FloodGrapher's plugin manager - just unzip it, open up FloodGrapher, go to the "Plugins" tab, hit the add button under the file format list, and type in "org.robochina.plugin.StatistRobotFormat?" in the box. Now if you were using StatistRobot for testing in the past, you can use FloodGrapher to view it now. -- Kawigi
