BlackWidow - Small, black, shiny and deadly (well, not) - by PEZ

What's special about it?

It's my first MicroBot. It can hardly hit the boundaries of the battle field.

Great, I want to try it. Where can I download it?

http://www.robocoderepository.com/BotDetail.jsp?id=1395

How competitive is it?

It ranks #70 in EternalRumble all weight classes and #18 amongst MicroBots. This is probably all due the movement it shares with Gouldingi and Marshmallow.

How does it move?

RandomMovement

How does it fire?

It remembers the angular movement of the enemy bot it has fired upon. About every third bullet. This angular movement is then somewhat randomized and used for aiming. Don't do this. BlackWidow proves how useless it is.

How does it dodge bullets?

No dodging here.

How does the melee strategy differ from one-on-one strategy?

It doesn't.

How does it select a target to attack/avoid in melee?

It doesn't choose. I guess it's useless gun is even more useless in Mellee?

What does it save between rounds and matches?

Nothing really. Though statics make less codesize so all bot variables are static and thus saved between rounds.

Where did you get the name?

Well, I don't like spiders very much and a BlackWidow is as deadly as spiders get.
