Spike's Challenge

This is the setup:
  • SandboxDT 1.91 in reference mode
  • The Challenger (your bot)
  • Arena size 1000 x 1000
  • 1000 rounds
  • Everything else set to Robocode defaults
Additionally:
  1. Challenger is not allowed to fire any bullets at all
  2. The ratio "Challenger score / reference bot score" * 100 is the Challenger's Index
  3. Please do not decompile or extend Spike ;)
