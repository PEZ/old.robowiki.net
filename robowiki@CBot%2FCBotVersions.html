CBot versions

Here are some of the different versions of CBot. To test the latest version download it from [Robocoderepository]

2003-02-17 CBot 0.63

  • New antigravity driver, far from finished
  • Fixed bug in 1v1 scanning
  • Added pattern matcher gun
  • Now only aimong when firing

2003-01-27 CBot 0.6

  • Totaly new gun with different strategies
  • 1v1 scanning
  • Driving still suck!
  • Some unit tests

2002-12-30 CBot 0.52

  • Fixed the detection when a pray i firing, now working mutch better
  • Made the fire detection event based
  • Tried to make pray manager more efficient
  • Some refactoring

2002-12-23 CBot 0.51

  • Wall avoidance. Don't hit the wall quite as often as before.
  • Wall collision move when CBot do hit a wall.
  • 1v1 mode with bullet dodging, not working very well.

2002-12-18 CBot 0.5

  • Best hitting rate so far
  • Just simple linear targeting
  • Simple oscillating moving pattern
  • No wall avoidance
