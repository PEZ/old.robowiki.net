JGAP

JGAP is an open source Java framework and can be found at http://jgap.sf.net.
Part of JGAP is RobocodeJGAP.

JGAP contains more than 1300 unit tests, a lot of examples and extendes Javadocs. Genetic Algorithms are the building blocks of JGAP, where Genetic Programming was added on top of that.

Project admin and lead developer is Klaus Meffert.
