Bot Name

Sprout

Author

Simonton

Extends

AdvancedRobot

What's special about it?

It's a pattern-matching micro melee bot. I don't know if there's many of those!

Great, I want to try it. Where can I download it?

Check RoboRumble/MeleeParticipants. I use whatever is actually available at the time I want to release.

How competitive is it?

We'll see!

How does it move?

It uses the movement of MeleeSeed, Sprout's baby brother.

How does it fire?

With a LateralVelocity pattern matcher, that interpolates between scans when needed.

How does it dodge bullets?

See MeleeSeed.

How does the melee strategy differ from one-on-one strategy?

It does not.

How does it select a target to attack/avoid in melee?

It attacks the closest target. It's movement is affected by the closest target, too, I suppose.

What does it save between rounds and matches?

Between rounds it saves pattern data on every enemy. Nothing between matches.

Where did you get the name?

MeleeSeed turns into Sprout. Makes sense to me.

Can I use your code?

RWPCL

What's next for your robot?

The number 1 spot in micro melee!

Does it have any WhiteWhales?

Any micro that stands in its way!

What other robot(s) is it based on?

The movement is from MeleeSeed, which is based a lot on DuelistNanoMelee. The gun is a standard nano pattern matcher (props go first to FunkyChicken, maybe?), and I think reading about TronGuns long ago is what came back to me to inspire interpolating between scans.
